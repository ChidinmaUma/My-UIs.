 # 1 create a main project folder

# 2 In main folder open terminal and run the packagejson cmd 
First cmd: npm init -y

# 3 create a newfile called README.md and two folders called public and src.

# 4 In public folder create (static resources files e.g images can be created inside the public)
.Images,Audio,Videos,Docs,HTML Files etc All resources always remain the same & always end with extension like (.html, .htm)

# 5 In Src (dynamic resources)
.css,scss,js,ts,less etc Responds according to different inputs just like filling a form & Always end with extension like (.asp,.jsp,.php,.aspx)

# 6 webPages re classified into 2. (The static & Dynamic)  both are designed using HTML and it could look same but the difference depends on how it responds when run.
